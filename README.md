2. แตกไฟล์ zip นี้แล้ว upload ทุกไฟล์ขึ้น repository
3. ไปที่ **Settings → Pages → Source: Deploy from branch → main**

5. รอประมาณ 2-5 นาที แล้วเปิด https://nextnovel.click
